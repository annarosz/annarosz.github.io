> Web design / development

The challenge

Building a custom wesbite using a CMS theme

Sunified uses the Hubspot platform as CRM (Customer Relationship Management), as well as a CMS (Content Management System). The website was to be built on this platform, using a theme template, while still being on-brand.

The solution

Working within and embracing limitations

The first step was choosing a theme that offered a wide range of components and the ability for customisation. Through thorough testing of the theme I was able to assess what was and was not able to be built as required. I bridged the gap by creating many custom components using HTML, CSS and some Javascript. There were multiple design iterations based on what was functionally possible with the build.
